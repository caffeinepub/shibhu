# Specification

## Summary
**Goal:** Build a single-page, responsive “ShiBhu - Thought Leadership Portfolio” website that matches the structure, content, and visual style of the provided HTML.

**Planned changes:**
- Create a single-page layout with sections: Header, sticky Navigation, Articles, Core Expertise, Career Overview/Stats, About, Footer.
- Implement the Header with the provided name, tagline, and five expertise tags rendered as wrapping pill/badge elements on a dark gradient background.
- Implement a sticky top Navigation bar with links to #articles, #expertise, #stats (Portfolio), and #about, including accessible hover/focus states and smooth section scrolling.
- Build the Articles section with the provided category headings and a responsive article-card grid; each card shows title, platform badge, date, description, and an outbound link opening in a new tab with proper rel attributes.
- Build the Core Expertise section as a responsive grid of cards/blocks with the provided titles/descriptions and a clear accent treatment.
- Build the Career Overview section showing the four provided stats in a responsive grid of uniform stat cards.
- Build the About section with the provided two-paragraph bio, centered with readable max width and typography.
- Build the Footer with the provided copyright line and “Portfolio Last Updated: February 15, 2026”, styled with dark background and light centered text.
- Apply a consistent Tailwind theme inspired by the provided CSS (dark hero gradient, off-white surfaces, light-gray section backgrounds, subtle borders, single accent color similar to #0066cc; no blue+purple dual theme).

**User-visible outcome:** Users can view a polished, mobile-friendly single-page portfolio and use the sticky navigation to jump between sections, browse categorized article cards, and open article links in new tabs.
